TypeScript/JavaScript에서 `!`는 문맥에 따라 3가지 의미가 있다.

---

## 1) `!expr` : 논리 NOT (JavaScript)
- 표현식의 truthy/falsey를 뒤집어 boolean으로 만든다.
- 결과 타입: `boolean`
- 예시
  - `!null` -> `true`
  - `!"abc"` -> `false`
  - `!0` -> `true`
 
---

## 2) `!!expr` : boolean 정규화 (JavaScript)
- `!expr`를 두 번 적용해 truthy/falsey를 `true/false` 값으로 변환한다.
- 결과 타입: `boolean`
- 예시
  - `!!null` -> false
  - `!!"abc"` -> true
  - `!!0` -> false

---

## 3) `expr!` : Non-null assertion (TypeScript)
- 컴파일러에게 "null/undefined가 아님"을 단언한다.
- 런타임 동작 변화 없음 (값을 바꾸지 않음)
- 가정이 틀리면 런타임에서 오류가 날 수 있음
- 예시
  - `document.getElementById("root")!`
- 주의
  - 안전장치가 아니라 타입 검사 우회이므로 100% 확실할 때만 사용해야 한다.
